Requirements
------------

1. Linux
1. git
1. Python >= 3.5
1. colorama for python 3 (use pip)
1. [SDCC](http://sdcc.sourceforge.net/snap.php)
1. [stm8flash](https://github.com/vdudouyt/stm8flash)

Quick start
-----------

* `$ git clone https://github.com/davide99/stm8.git`
* `$ cd stm8`
* `$ sudo chmod +x make.py`
* `$ ./make.py flash`
* Done
